# HRMS Frontend - React Application

A modern Human Resource Management System (HRMS) frontend built using React.js. This application communicates with a Spring Boot REST API to perform CRUD operations for Departments and Employees.

## Features

- Department Management
  - Add Department
  - View All Departments
  - Update Department
  - Delete Department

- Employee Management
  - Add Employee
  - View All Employees
  - Update Employee
  - Delete Employee

- Responsive UI using Bootstrap

- REST API Integration using Axios

- Client-side Routing with React Router

## Installation

Clone the repository

bash
git clone https://github.com/vikkash-j/Hrms-Frontend.git


Navigate to the project

bash
cd Hrms-Frontend


Install dependencies

bash
npm install


Run the application

bash
npm start


The application will start on


http://localhost:3000
